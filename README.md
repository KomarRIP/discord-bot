## Discord бот-конструктор милсим подразделения (Arma 3, стиль ССО РФ)

Этот репозиторий содержит **архитектурное ядро** проекта: доменную модель, границы модулей, схемы конфигов и дорожную карту.
Цель продукта — после приглашения бота в пустой Discord‑сервер провести **Setup Wizard** и автоматически развернуть полностью готовую структуру: роли, каналы, права, приёмную/пропускной режим, журналы и правила.

### Статус

- **Фаза 0 (архитектура/домен)**: описано в `docs/core.md`.
- **MVP шаблон**: `SSO_RF` (первый внедряемый шаблон).

### Ключевые принципы

- **Think first, code later**: сначала фиксируем модель и контракт модулей, затем реализация.
- **Config-driven**: шаблоны и права задаются конфигами, логика — универсальная.
- **Идемпотентный деплой**: повторный запуск не создаёт дубликаты, а приводит сервер к целевому состоянию.
- **Строгие границы модулей**: отсутствие «бог‑сервисов», чёткие зависимости.

### Документация

- `docs/core.md` — единый источник правды: обзор, глоссарий, архитектура, домен, схемы конфигов, roadmap, конвенции.
